Java Intro

Java is a high-level programming language originally developed by Sun Microsystems and released in 1995. Java is platform independent,platforms such as Windows, Mac OS, and the various versions of UNIX can easily configure Java Virtual Machine(JVM) and run java app. The latest release of the Java Standard Edition is Java SE 8.
Key features of java -

Object Oriented - In java every thing is an Object. Java Object models can easily extended to create custom libraries
Platform Independent/Protable - Java is compiled into platform independent byte codes. The byte codes can be distributed and iterpreted by the JVM on whicherver platform its beging running on
Secure - Java virtual machine manage the access to system resource and application codes. admins can easly restrict any unwanted resouce access
Developer Community - Java is open source and backed by strong developer community. There are lot of open source tools that eage the developers life most
Well Documented - Java is one of the most well documented language. All the stadard libraries have java docs that allow developers too kick start developement
